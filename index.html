Section One

Forms and Links

Here I Would like to create a form, fill it and link the submission to a location in another webpage. By so doing, I get to practice HTML forms and links

To build a form, these are the necessary steps

  • Step One

    • Put in a form tag, usually, a form is accompanied with the action="" attribute. It's function is basically to submit the form to a certain page, say, a php page. A form tag also has the method="" attribute. It is the POST for the form, i.e, it is used to make a post request. Posts are a secure way of adding data to a database.

  • Step Two

    • Next, create a label tag, after the label tag create an input tag. The label is used to indicate the Label of the form, eg. First Name. The input tag on the other hand has the type="" attribute in which the type of input needed is indicated. Also the input tag has a name="" attribute.

  • Step Three

    • Note: When building a form for with an email input, the type="" attribute should be 'email', not 'text'. Again, if you want to create a text area in the form, say a message or comment box, use the textarea tag instead of the label tag, and if you are creating a select list like gender, Marital Status, etc., use the select tag.
    Each select list has a option tag, the option tag has a value="" attribute where you can input the value (for the server) and the actual text goes between the option tag.
    When creating an age form, the type="" attribute is 'number' while the value=" attribute indicates the range.
    A birthday label has a date type="" attribute.

    N|B The placeholder="" attribute is used to attach a faint indicator in the form.

  • Step Four

    • The last thing needed when building a form is the button input. create an input tag, with the type="" attribute (submit), with name="" Submit and Value="" submit which is the actual text of the button eg. Submit, Alldone etc.
    Use the button tagto indicate the some sort of function like 'click me' and then use CSS to link it to another site.
